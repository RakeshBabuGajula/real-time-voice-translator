# Contributing to Real-Time Voice Translator

Thank you for your interest in contributing to this project! We welcome contributions from the community to improve the app.

## How to Contribute

1. **Fork the repository** and clone your fork locally.
2. **Create a new branch** for your feature or bugfix:
   ```
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** in the new branch.
4. **Test your changes** thoroughly.
5. **Commit your changes** with clear and descriptive messages.
6. **Push your branch** to your fork:
   ```
   git push origin feature/your-feature-name
   ```
7. **Open a Pull Request** on the main repository.

## Code Style

- Follow PEP 8 style guidelines for Python code.
- Write clear, readable, and maintainable code.
- Include comments and docstrings where appropriate.

## Testing

- Test new features and bug fixes thoroughly.
- Ensure existing functionality is not broken.

## Reporting Issues

- Use the GitHub Issues tracker to report bugs or request features.
- Provide clear and detailed descriptions, including steps to reproduce.

## Communication

- Be respectful and constructive in discussions.
- Follow the project's code of conduct.

Thank you for helping improve the Real-Time Voice Translator project!
