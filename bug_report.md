# Bug Report

## Description
In the Streamlit run server, the transcribed recorded audio player is displayed too lightly, making it difficult to see clearly.

## Steps to Reproduce
Steps to reproduce the behavior:
1. Run the Streamlit app using `streamlit run app_streamlit.py`.
2. Select input and output languages.
3. Start the translation and speak into the microphone.
4. Observe the transcribed audio player displayed below the translated text.

## Expected behavior
The transcribed audio player should be clearly visible with appropriate contrast and styling to ensure usability.

![Audio Player Visibility Issue](./.gitlab/issue_templates/audio_player_issue.png)

## Additional context
This issue affects user experience by making it hard to interact with the audio player controls.
